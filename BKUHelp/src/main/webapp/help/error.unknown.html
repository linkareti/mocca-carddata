Unerwarteter Fehler

Ein unerwarteter Fehler ist aufgetretten. Bitte wiederholen Sie den Vorgang noch einmal.

Falls Sie den Fehler melden wollen, ist es sehr hilfreich wenn Sie das Log-File mitschicken.

Bei lokale BKU (MOCCA Webstart) finden Sie das Log-File unter dem Home-Verzeichnis des Benutzers (unter Windows %USERPROFILE%\.mocca\logs\webstart.log)

  • z.B. C:\Benutzer\tester\.mocca\logs\webstart.log

Verwenden Sie online BKU MOCCA so müssen Sie zuerst die Java Protokollierung aktivieren. Unter Windows finden Sie die Java Einstellungen unter Systemsteuerung -> Java Control Panel -> Erweitert -> Debugging -> Protokollierung aktivieren.

Damit das Log-File auch angezeigt wird, ist es notwendig die Java-Konsole einzublenden. Dies können Sie ebenfalls im Java Control Panal einlstellen. Java Control Panel -> Erweitert -> Java - Konsole -> Konsole einblenden.

Das Java Applet Log-Level kann weiters noch feiner eingestellt werden. Diese Konfiguration können sie unter JAVA_HOME/lib/logging.properties einstellen.

  • z.B. unter Windows C:\Programme\Java\jre6\lib\logging.properties

Hierbei ist es wichtig, dass das Logging auf FINEST eingestellt wird, wie unten in der mittleren Zeile gezeigt wird.

# Limit the message that are printed on the console to INFO and above. java.util.logging.ConsoleHandler.level = FINEST java.util.logging.ConsoleHandler.formatter = java.util.logging.SimpleFormatter

Weiter Informationen finden sie unter: Logging und Tracing
